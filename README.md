# TOWA slider

Create a slider using web components

## Installation

To use the right Node version on your laptop run:
```bash
npm run nvm-use
```

Afterwards use the following command:
```bash
npm i
```

## Usage

For development usage:
```bash
npm run dev
```

For production usage:
```bash
npm run build
```

To preview the production build:
```bash
npm run preview
```
